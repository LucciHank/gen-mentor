# GenMentor Core UI System

Bộ 13 page hoàn chỉnh: 8 màn hình lõi + 5 lớp UI phụ

## 🎯 Nguyên tắc UX

1. **Mỗi màn chỉ có 1 việc chính**
2. **Không tạo page mới nếu drawer đủ dùng**
3. **Agent chỉ xuất hiện khi có ích**
4. **Dùng 1 layout chung cho 3 nhóm user**
5. **Apple HIG style - sạch, tối giản**
6. **"Hôm nay học gì" là trung tâm**

## 📱 8 Màn hình lõi

1. **Dashboard / Hôm nay** - Biết hôm nay học gì
2. **Tạo lộ trình** - Progressive form 1 page
3. **Roadmap** - Nhìn toàn cảnh
4. **Chi tiết chặng / Module** - Kết nối roadmap với học thật
5. **Lesson / Video player** - Tập trung học
6. **Task / Kế hoạch tuần** - Thực thi
7. **Tiến độ / Kỹ năng** - Xem tiến độ
8. **Review & Điều chỉnh** - Điều chỉnh kế hoạch

## 🔧 5 Lớp UI phụ

1. **AI Coach Side Panel** - Trợ lý ngữ cảnh
2. **Floating Mini Planner** - Luôn biết việc cần làm
3. **Reschedule Drawer** - Dời lịch nhanh
4. **Quick Review Sheet** - Feedback sau lesson
5. **Notification Panel** - Thông báo từ topbar

## 👥 3 Nhóm User

- **Học viên ngoài** - Goal cá nhân, tự chọn pace
- **Nhân viên đang làm** - Task bắt buộc, due date
- **Người mới vào công ty** - Onboarding checklist

## 🎨 Design System

- **Nền sáng, spacing rộng**
- **Card ít, border nhẹ, shadow rất nhẹ**
- **Không gradient, 1 màu chủ đạo**
- **Chữ rõ, CTA rõ**
